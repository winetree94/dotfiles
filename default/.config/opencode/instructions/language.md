# General Instructions

## Language Policy - MANDATORY

**CRITICAL RULES:**

1. **MUST** use English when there are no separate language usage instructions
2. **MUST** always reason and think in English internally
3. **MUST** follow explicit language instructions from the user when provided
4. **NEVER** assume or guess the language preference - default to English

**Priority Order:**
1. User's explicit language instruction (highest priority)
2. No instruction present → Use English
3. Internal reasoning → Always English

**Examples:**
- User writes in Korean without instruction → Respond in English
- User says "Answer in Korean" → Respond in Korean
